PROJECT 02COMMUNICATION PROTOCOL
차세대 통신 프로토콜 전환 UART → CAN

추진 목적 및 기술적 당위성

기존 1:1 통신 방식인 UART 프로토콜은 배터리, 모터 컨트롤러, 블랙박스 등 차량 내 멀티 노드 증가 시 노이즈 취약 및 배선 한계 직면. 버스 구조의 차동 신호 방식을 채택하는 **CAN(Controller Area Network) 통신 시스템**으로 전면 전환하여 하드웨어 제어 안정성 확보와 고장 진단 모듈(OBD) 확장 토대를 마련했습니다.

수행 성과

✓전체 단말 패킷 에러율(CRC Error) 0.02% 미만으로 절감
✓실시간 배터리 셀(Cell) 상태 수집 주기 단축 및 예측 기반 BMS 관리 기능 구현

PROJECT 03BILLING ENGINE LOGIC
GPS 기반 하이브리드 과금 엔진 로직 설계

해결하고자 지향한 목표

도심지 음영 지역(빌딩 숲 등)에서 GPS 튀는 현상으로 인해 오과금이 발생하거나 사용자 반발 클레임이 다수 접수되는 문제 직면. 단말의 가속도 센서 지표와 통신 기지국 기반 위치값(LBS), 라우트 스냅 알고리즘을 융합한 보정 체계를 구축해 과금 오류율을 근본적으로 극복했습니다.
